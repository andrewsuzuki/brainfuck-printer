# brainfuck-printer

Simple command line tool to create Brainfuck programs that print strings.

## Usage

    ruby run.rb "hello world"
    # Output: ++++++++++[>++++++++++>+++<<-]>++++.---.+++++++..+++.>++.<++++++++.--------.+++.------.--------.
